## ğŸ“± Mobile Responsive Features

### Optimizations for Mobile:
- **Hero Section**: Reduced padding, smaller fonts, repositioned decorations
- **Countdown Timer**: Responsive grid (4 columns on tablet, 2 columns on small mobile)
- **Gallery**: Single column layout with taller image containers (400px tablet, 350px mobile)
- **Image Display**: Uses `object-fit: contain` to prevent cropping of heads/faces
- **Navigation**: Touch-friendly button sizes and spacing
- **Forms**: Mobile-optimized input fields and buttons
- **Shehnai Decorations**: Hidden on extra small screens (<480px) for cleaner look
